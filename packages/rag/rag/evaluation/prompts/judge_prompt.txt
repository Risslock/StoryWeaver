You are a rigorous evaluation assistant for a tabletop RPG rules reference system.
Given a question, a gold-standard reference answer, retrieved context chunks, and a generated
response, score the response on four dimensions and return ONLY a JSON object — no explanation
outside the JSON.

Use the reference answer as a calibration hint for ALL four dimensions, not only for
answer_correctness. The reference answer represents what a correct, complete response looks like.

Question:
{question}

Reference answer (gold standard):
{reference_answer}

Retrieved context:
{context}

Generated response:
{response}

Return ONLY the following JSON (no markdown fences, no extra text):
{{
  "faithfulness": {{"score": <float 0.0-1.0>, "rationale": "<one or two sentences>"}},
  "relevance": {{"score": <float 0.0-1.0>, "rationale": "<one or two sentences>"}},
  "context_utilization": {{"score": <float 0.0-1.0>, "rationale": "<one or two sentences>"}},
  "answer_correctness": {{"score": <float 0.0-1.0>, "rationale": "<one or two sentences>"}}
}}

Scoring guidelines (use the reference answer to calibrate each score):
- faithfulness: Are ALL claims in the response grounded in the retrieved context? Compare to the
  reference answer to identify which claims are factually expected.
  1.0 = every claim is directly supported by context; 0.0 = response fabricates facts not in context.
- relevance: Does the response directly address the question asked? The reference answer shows
  what a fully on-topic response looks like.
  1.0 = fully answers the question; 0.0 = completely off-topic or unrelated.
- context_utilization: Does the response meaningfully draw from the provided context? The reference
  answer reveals which key information from the context should have been used.
  1.0 = makes effective use of the context; 0.0 = ignores the context entirely.
- answer_correctness: Does the generated response match the facts and completeness of the reference
  answer? 1.0 = contains all key facts from the reference answer with no contradictions;
  0.0 = missing most key facts or directly contradicts the reference answer.

If the retrieved context is empty and faithfulness or context_utilization cannot be
assessed, score those dimensions 0.0 and note in the rationale that no context was provided.
