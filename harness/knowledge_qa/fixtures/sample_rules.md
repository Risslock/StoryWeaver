# Earthdawn 4E Sample Rules

## Combat Initiative

At the start of combat, all participants roll Initiative using their DEX step.
The character with the highest result acts first. In the event of a tie, the
character with the higher DEX attribute acts first. If still tied, compare
Initiative dice results; ties are broken simultaneously.

## Talents

A Talent is a special ability or skill that a character learns through their
Discipline. Each Talent has a Rank (1–15) that determines its step number when
used. Talents are the primary way Adepts express their magic and combat prowess.

### Using a Talent

To use a Talent, a character rolls the step number associated with that
Talent's Rank and adds any relevant attribute modifier. The result is compared
against a Difficulty Number (DN). Meeting or exceeding the DN is a success.

## Difficulty Numbers

The Gamemaster sets the Difficulty Number for tasks:

| Task Difficulty | DN |
|---|---|
| Simple          |  5 |
| Average         |  7 |
| Challenging     | 10 |
| Hard            | 13 |
| Very Hard       | 16 |
| Heroic          | 20 |
| Legendary       | 25 |

## Karma

Karma points allow Adepts to add an extra die roll to any test. The Karma die
depends on the Discipline. Spending Karma is optional and costs one Karma point
per use. Karma pools refresh at the start of each session or after a night's rest.
